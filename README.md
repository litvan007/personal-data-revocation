# Отзыв персональных данных: гайд и контакты

## Общая процедура (152-ФЗ)

### Что такое отзыв ПДн
Субъект персональных данных вправе в любой момент отозвать согласие на их обработку (ст. 9 ФЗ-152). После отзыва оператор обязан прекратить обработку и уничтожить данные в срок **до 30 дней** (ст. 21).

### Исключения — что не удалят
- Данные, необходимые для исполнения договора (пока договор действует)
- Данные, которые оператор обязан хранить по закону (бухгалтерия, налоги — 3-5 лет)
- Данные в обезличенном виде (для статистики)

### Шаблон заявления

```
Кому: [Название компании]
Юридический адрес: [адрес]
DPO / Ответственный за обработку ПДн: [ФИО, если известен]

От: [ФИО полностью]
Адрес: [адрес регистрации]
Паспорт: серия №, выдан [кем, когда]
Телефон: [+7...]
Email: [...]

ЗАЯВЛЕНИЕ
об отзыве согласия на обработку персональных данных

Настоящим заявлением отзываю свое согласие на обработку моих персональных данных,
ранее данное [когда/где — например: "при регистрации на сайте ..."].

Прошу:
1. Прекратить обработку моих персональных данных
2. Уничтожить мои персональные данные в сроки, установленные законодательством
3. Подтвердить факт прекращения обработки и уничтожения ПДн письменно

Мои данные в вашей системе:
- Аккаунт / логин: [...]
- Email: [...]
- Телефон: [...]

Приложение: копия паспорта (основная страница)

Дата: [ДД.ММ.ГГГГ]
Подпись: ___________
```

### Как отправить
1. **Заказным письмом** с уведомлением о вручении на юридический адрес — самый надёжный способ
2. **Электронно** — на email DPO или через форму обратной связи (если есть)
3. **Через Госуслуги** — для госорганов (не для коммерческих сервисов)

## Общий чеклист

1. Сначала попробуй удалить аккаунт через настройки приложения/сайта
2. Если нужно точечно отозвать только маркетинговые данные — напиши в support
3. Для полного юридического отзыва — заказное письмо на юр. адрес с заявлением и копией паспорта
4. Сохрани квитанцию отправки + уведомление о вручении
5. Если нет ответа 30 дней — жалоба в Роскомнадзор (rkn.gov.ru)
